Functions Of My Calculator

  • It can perfrom all the arithmetic Operations.
    1. Addition (+)
    2. Subtraction (-)
    3. Multiplication (*)
    4. Division (/)
    5. Modulo (%) - It gives the remainder of a division.
  • It can perform Trigonometric Operations.
    1. sin(x) function
    2. cos(x) function
    3. tan(x) function

How to Use this calculator

  1. To apply negative sign to number use " (- ".
  2. You can perform any Arithmetic operations
    • Input the first Operand.
    • Select Which operation to perform.
    • Input the second operand.
  3. For using Trigonometric Functions
    • First Select the function from dropdown.
    • Then Input the value in Degree.
